# 宝塔Linux面板 For Docker

## 运行
```
docker run -p 21:21 \
           -p 80:80 \
           -p 443:443 \
           -p 888:888 \
           -p 3306:3306 \
           -p 8888:8888 \
           -p 39000-40000:39000-40000 \
           -d --name bt \
           mingfunwong/bt
```

